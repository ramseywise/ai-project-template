# What grind size and dose should I use for espresso?

Use a fine grind, similar to table salt — too coarse and the shot will run too fast
and taste sour; too fine and it will choke the machine. Start with an 18g dose for a
double shot, aiming for a 1:2 ratio (18g in, 36g out) over 25-30 seconds. Adjust
grind size in small steps and re-taste before changing dose.
